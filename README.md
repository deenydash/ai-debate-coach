# AI Debate Coach

AI Debate Coach is an interactive web app that helps users improve debate and argumentation skills. The app generates counterarguments, scores the user's argument from 0–10, and provides coaching tips to improve reasoning and persuasion. Users can switch between Coach, Opponent, and Judge modes and optionally enable voice narration for AI responses.

------------------------------------------------------------

Features

• Debate Modes: Coach, Opponent, or Judge.
• Score System: Each user argument receives a 0–10 score.
• Coaching Tips: Short, targeted feedback to strengthen arguments.
• Conversation Memory: AI remembers discussion context.
• Voice Mode: AI can speak responses aloud (On/Off toggle).
• Sidebar Display: Shows topic, mode, and average score.

------------------------------------------------------------

Tech Stack

• React (JavaScript) - User Interface
• Google Gemini API - Argument generation and scoring
• Web Speech API - Text-to-speech voice playback
• CSS - Styling and layout

------------------------------------------------------------

Setup Instructions

1. Clone the repository:
   git clone https://github.com/deenydash/ai-debate-coach.git
   cd ai-debate-coach

2. Install dependencies:
   npm install

3. Create a .env file in the project folder and add:
   REACT_APP_GEMINI_API_KEY=YOUR_API_KEY_HERE

4. Start the application:
   npm start

App will run at:
http://localhost:3000

Why This Project Matters

This project helps users build confidence in debate by improving structure, clarity, and reasoning. By receiving immediate feedback and counterarguments, users develop stronger argumentative and critical thinking skills.
